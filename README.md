# NIELIT TRACK
NIELIT TRACK Attendance Manager application for Android developed using Java, SQLite and Shared Preferences.

## Author
- Abhay Singh and Manjinder Kaur

## Features
- Add Student
- View Student
- Add Faculty
- View Faculty
- Manage Attendance

## Usage
- Admin Username : `admin`
- Admin Password : `pass`
